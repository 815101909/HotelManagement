# 酒店管理系统

## 项目简介
本项目为一套基于前后端分离的酒店管理系统，涵盖员工管理、客户管理、财务报表、仪表盘等核心业务模块，支持数据的增删改查、统计分析及导出等功能，适用于中小型酒店的信息化管理。

## 主要功能模块
- **仪表盘**：展示本月收入、入住率等关键运营数据。
- **员工管理**：员工信息的增删改查，支持编辑、删除、添加员工，数据与后端实体类字段严格对应。
- **客户管理**：客户信息管理，支持数据导出为Excel。
- **财务报表**：ECharts可视化展示收入、支出等财务数据，支持tooltip悬停提示。
- **权限设置**：多角色权限分配。
- **通知设置**：支持邮件、短信等通知方式配置。
- **备份与恢复**：系统数据备份与恢复操作。

## 技术栈
- **前端**：Vue 3、Axios、ECharts、Element Plus（或自定义UI）
- **后端**：Spring Boot、MyBatis、EasyExcel、MySQL

## 运行方式
### 后端启动
1. 配置`application.yml`数据库连接。
2. 执行`mvn clean install`，运行`HotelManagementApplication.java`。

### 前端启动
1. 进入`src/main/resources/static/admin`目录。
2. 安装依赖：`npm install`
3. 启动开发环境：`npm run serve` 或 `npm run dev`

### 访问系统
- 默认地址：http://localhost:8080/admin/

## 前后端对接说明
- 前端通过Axios请求后端RESTful接口，如`/employee/all`、`/employee/add`、`/api/customers/export`等。
- 前端表单字段需与后端实体类字段一一对应。
- 数据导出接口返回`blob`类型，前端自动下载Excel。
- 仪表盘等统计接口需传递日期参数，确保数据准确。

## 常见问题
- **接口404/405**：检查前后端接口路径、方法是否一致。
- **图片/图标不显示**：确认路径写法，public目录下资源应以`/资源名`访问。
- **ECharts tooltip不显示**：检查图表初始化与销毁逻辑。
- **表格列宽/按钮样式问题**：可通过CSS自定义调整。
- **git push超时**：检查网络、代理、DNS设置。

## 功能模块

### 基础管理
- **管理员信息管理**：个人信息维护、密码修改
- **员工管理**：员工信息录入、权限分配、工作排班

### 房间管理
- **房型管理**：不同类型房间的配置与价格设置
- **房间管理**：房间状态监控、设施维护记录
- **房间状态日历**：可视化日历视图，颜色区分房间状态（红-占用/绿-空闲）

### 预订与入住
- **预订管理**：在线预订、电话预订记录、预订变更
- **入住管理**：客人登记、房间分配、证件信息录入
- **退房管理**：结算账单、房间状态更新

### 客户管理
- **客户信息管理**：客户档案、消费记录、偏好设置
- **会员管理**：会员等级、积分规则、优惠政策

### 财务管理
- **收入记录**：房费收入、额外服务收入
- **支出管理**：运营成本、物资采购、工资支出
- **财务报表**：日报、月报、年度财务分析

### 统计分析
- **营收统计**：每日/月度/年度营收数据
- **入住率统计**：房型入住率、季节性分析
- **客源分析**：客户来源、年龄分布、消费习惯

## 安装部署

### 环境要求
- JDK 11+
- Node.js 14+
- MySQL 8.0+
- Maven 3.6+

### 后端部署
1. 克隆代码仓库
   ```bash
   git clone https://github.com/yourusername/hotel-management.git
   ```

2. 配置数据库
   - 创建名为 `db_hotel` 的数据库
   - 修改 `application.yml` 中的数据库连接信息

3. 构建并运行后端
   ```bash
   cd hotel-management
   mvn clean package
   java -jar target/hotel-management-0.0.1-SNAPSHOT.jar
   ```

### 前端部署
1. 进入前端目录
   ```bash
   cd src/main/resources/static/admin
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 开发模式运行
   ```bash
   npm run dev
   ```

4. 构建生产版本
   ```bash
   npm run build
   ```

系统使用MySQL数据库，主要表结构包括：

- **admin**：管理员信息表
- **employee**：员工信息表
- **room_type**：房型表
- **room**：房间表
- **booking**：预订信息表
- **checkin**：入住登记表
- **customer**：客户信息表
- **finance**：财务记录表
- **revenue**：营收统计表
